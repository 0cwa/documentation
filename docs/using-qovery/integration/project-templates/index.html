Project Templates

Templates are pre-configured Dockerfiles and .qovery.yml configs that allow you to bootstrap a new application quickly.

#Template

#List

You can list all available templates with the command:

$ qovery template list
All templates available
NAME   | DESCRIPTION
hasura | Hasura is a production-ready GraphQL backend
gatsby | Gatsby is a fast modern site generator for React
...

#Use

To use one of them, run:

mkdir my_hasura_application
cd my_hasura_application
git init


qovery init -t hasura
explain this command

#Contributions

Any idea of a new template? Contributions are welcomed.

#Final word

Now, all you have to do is to commit and publish your repository to Github, Gitlab, or Bitbucket, to deploy your application!
